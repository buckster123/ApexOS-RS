## Security

- ‚úÖ **DONE ‚ÄĒ `read_file`/`list_dir` path confinement** ‚ÄĒ `tools.rs::confine()` is now the single FS-tool gate: reads confined to workspace + a small allowlist (EDK parts, `/sys`, `/proc/cpuinfo`) minus a secret denylist (`/proc/*/environ`, `/etc/agentd/env`, `~/.ssh`, `/etc/shadow`, `*.api_key`); writes/deletes are workspace-only. `/proc/self/environ` token exfil is dead. **[was high]**
- ‚úÖ **DONE ‚ÄĒ `http_fetch` SSRF protection** ‚ÄĒ already had `ssrf_guard` (resolves host, blocks loopback/link-local/RFC1918) + a streaming 4MB cap (not post-buffer). This PR adds a redirect policy that re-runs the guard on each hop. Residual: DNS-rebind TOCTOU (needs a pinned-IP connector). **[was high ‚Üí low residual]**
- ‚úÖ **DONE ‚ÄĒ Command injection via `bootstrap_node`** ‚ÄĒ partially mis-scoped in the original finding: `ssh_password`/`repo_url`/`api_key` were already `shell_single_quote`'d (the `{:?}` was gone). Tests: `bootstrap_node_rejects_injection_in_user_and_host`. The sibling vast.ai tunnel SSH (`:1311`) is argv-only (no shell) with API-sourced host/port ‚Üí not the same vector. **[was high]**
